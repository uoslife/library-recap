# [TF] 도서관 Recap 이벤트: client

### 소개
- 시대생 유저의 2023년도 학교 도서관 이용내역을 스토리 형식으로 보여주고, Instagram 리그램 이벤트를 진행합니다.
- 시대생 client 앱내 웹앱을 통해 이용 내역을 보여줍니다.

### 배포 링크
